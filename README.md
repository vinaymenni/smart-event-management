# Smart Event Management System

## 📌 Overview
Smart Event Management System is a web-based application developed using Spring Boot that allows users to register for events and enables administrators to manage events efficiently.

## 🚀 Features
- User Registration & Login
- Role-based Access (Admin / Student)
- Event Creation and Management
- Secure Backend using Spring Boot
- Database integration using H2

## 🛠️ Tech Stack
- Java
- Spring Boot
- Maven
- H2 Database
- Hibernate (JPA)

## 🏗️ Architecture
- Controller Layer
- Service Layer
- Repository Layer
- Model (Entity) Layer

## ▶️ How to Run
1. Clone the repository
2. Open in Eclipse or IntelliJ
3. Run `SmartEventsApplication.java`
4. Open browser at `http://localhost:8080`
